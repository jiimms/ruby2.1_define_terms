Define the following

  • Object

    • Everything in Ruby is an object. The activities that an object performs defines its behaviour

  • Class

    • Class is a representation of type of object. Class consists of names, attributes and operations

  • Inheritance

    • Inheritance is where an object or class inherits or shares properties from its parent object or class. This means it acquires the same properties and methods of it's parent.

  • Polymorphism

    • Polymorphism is the ability to take more than one form. Polymorphism is using the same method on different objects and getting different results.

  • Encapsulation

    • Encapsulation means binding the data and function together. Encapsulation is the packing of data and functions into a single component. Encapsulation means that the internal representation of an object is hidden from the outside.

  • Abstraction

    • Abstraction means hiding the code from backend user. It is the primary means of managing complexity in large programs.
